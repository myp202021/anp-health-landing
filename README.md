# ANP Health Landing

Landing page profesional para ANP Health — validación de credenciales de enfermería para EE.UU.

**Live:** https://anp-health-landing.vercel.app

## Stack
- Next.js 15 + TypeScript + Tailwind CSS
- SEO: JSON-LD schemas (Organization, FAQPage, Service)
- Brand: Anko + DM Sans fonts, colores manual de marca (#18234b, #3d6c91, #73bcdb, #7ff2ff)
- Deploy: Vercel

## Secciones
- Hero con foto enfermeros + counters animados
- Pain Points (USA vs LATAM)
- Social Proof (+2,000 profesionales, 28 países)
- Flujo 3D interactivo 8 etapas
- Formulario calificación tipo GHL
- WhyANP (diferenciación competitiva)
- Benefits (salario, housing, sponsor)
- Video testimonios (6 slots YouTube)
- Blog (3 artículos SEO)
- FAQ 8 preguntas
- Schemas IA (knowsAbout, FAQPage)
